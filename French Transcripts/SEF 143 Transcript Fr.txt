Pour mettre un adjectif au féminin, on ajoute un -e,
2
ce qui modifie la prononciation de l’adjectif.
Il est français ; elle est française.
Il est allemand ; elle est allemande.
Il est chinois ; elle est chinoise.
Il est marocain ; elle est marocaine.
Il est argentin ; elle est argentine.
Pour les adjectifs masculins qui finissent par -n, on ajoute
un -n et un -e, ce qui change aussi la prononciation.
Il est italien ; elle est italienne.
Il est guinéen ; elle est guinéenne.
Il est brésilien ; elle est brésilienne.
Il est indien ; elle est indienne.
Certains adjectifs de nationalité ne changent pas
entre le masculin et le féminin.
Il est russe ; elle est russe.
Il est belge ; elle est belge.
Il est croate ; elle est croate.
Au pluriel, on ajoute un -s, ce qui ne change pas la prononciation.
Il est roumain ; ils sont roumains.
Il est vietnamien ; ils sont vietnamiens.
Pour dire le pays d’où l’on vient, on utilise
“du” avec les noms de pays masculins,
3
“de” avec les noms de pays féminins et les îles,
“des”, avec les noms de pays pluriels,
et “d’” avec les noms de pays commençant par une voyelle.
Le Portugal. Elle vient du Portugal.
Le Liban. Il vient du Liban.
La Pologne. Elles viennent de Pologne.
La Corée du Sud. Ils viennent de Corée du Sud.
Madagascar. Elle vient de Madagascar.
Taïwan. Il vient de Taïwan.
Les Etats-Unis. Elles viennent des Etats-Unis.
Les Philippines. Ils viennent des Philippines.
L’Espagne. Elle vient d’Espagne.
L’Uruguay. Il vient d’Uruguay.
